Linux常用命令

# netstat

netstat用于显示网络状态

netstat -ntlp # 查看当前所有tcp端口
netstat -an | grep 80 # 查看80端口使用情况

# curl

curl是常用的命令行工具，用来请求Web服务器。它的名字就是客户端（client）的URL工具的意思。

不携带任何参数时，curl发出GET请求

curl https://www.google.com

-d参数用于发送POST请求

curl -d "opr=pwdLogin&userName=用户名&pwd=密码&rememberPwd=1" https://www.google.com/login

# ufw

ufw是Ubuntu系列发行版自带的类似iptables的防火墙管理软件，底层也是基于netfilter的。

开启/关闭防火墙

sudo ufw enable  # 开启防火墙
sudo ufw disable # 关闭防火墙

查看防火墙状态

$sudo ufw status

开放端口

sudo ufw allow 80
sudo ufw allow 80/tcp # 开发80端口，特定协议

重启防火墙

sudo ufw reload

访问

sudo ufw allow from 192.168.1.1 # 允许特定ip访问所有端口
sudo ufw allow from 192.168.1.1 to any port 80 # 允许特定ip访问特定端口
sudo ufw delete allow from 192.168.1.1 # 移除特定ip访问所有端口

sudo default deny # 拒绝外部所有访问
sudo deny 80 # 拒绝端口访问
sudo delete deny 80 # 移除端口访问限制
