JDK安装指南

JDK是开发java程序必备的开发工具包，要想做java程序开发，那么就需要装JDK。

以往的java开发都是在win10上进行，由于项目的需要，转到了Linux系统上。

本篇记录一下在Ubuntu 20.04系统上配置JDK的过程。

# 1.下载安装包

jdk8 下载路径: 官方链接 (opens new window)

jdk11 下载路径: 官方链接 (opens new window)

jdk17 下载路径: 官方链接 (opens new window)

或者直接通过软件源安装

sudo apt-get update
sudo apt install openjdk-11-jdk

# 2.配置系统变量

sudo vim ~/.bashrc

#添加如下配置

# jdk-11
export JAVA_HOME=/usr/lib/jvm/java-11-openjdk-amd64
export JRE_HOME=${JAVA_HOME}/jre
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib
export PATH=${JAVA_HOME}/bin:$PATH

# 修改生效
source ~/.bashrc

# 测试是否安装成功
java -version

shell命令行输出jdk相关信息，证明jdk安装完成
